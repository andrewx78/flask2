## Развертывание на локальной машине 

1. Создаем вируальное окружение: 'python3 -m venv flask_venv'
2. Активируем venv: 'source flask_venv/bin/activate'
3. Устанавливаем зависиомсти: 'pip install -r requirements.txt'
4. Создаем локальную БД: 'flask db upgrade'
5. Запускаем приложение: 'puthon run.py'
